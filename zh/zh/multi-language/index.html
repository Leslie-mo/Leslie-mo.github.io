Available Languages

目前主题支持这些语言:

  • 英语
  • 简体中文

Configure languages

跟随 this Hugo documentation page 给你的网站配置语言。

Translation File Example

[blog]
one = "blog"
other = "blog"

[tags]
one = "tag"
other = "tags"

[archive]
one = "archive"
other = "archive"

[series]
one = "series"
other = "series"

[reading_time]
one = "One minute"
other = "{{ .Count }} min"

[word_count]
one = "one word"
other = "{{ .Count }} words"

[related_resources]
other = "Related Resources"

[featured_posts]
one = "Featured Posts"
other = "Featured Posts"
